About This Project

The Smart Home Automation System is an AI-powered computer vision project designed to automate indoor lighting based on real-time human presence detection. Instead of relying on traditional motion sensors, the system uses a YOLO-based deep learning model to detect occupants directly from camera input. The model was trained and fine-tuned on a custom indoor dataset and deployed on a Raspberry Pi, enabling fully on-device inference with low latency and improved privacy.

The project follows an end-to-end computer vision pipeline, including data annotation, preprocessing, model training, validation, and edge deployment.
