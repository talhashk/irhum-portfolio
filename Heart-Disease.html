Featured Project

Exploring Heart Disease Patterns with AI✨

This project focuses on using unsupervised machine learning techniques to uncover meaningful patterns within a heart disease dataset. By applying clustering and dimensionality reduction methods, the goal is to identify natural groupings of patients based on their health attributes.

Technologies Used

Python Pandas NumPy Matplotlib Seaborn Scikit-learn Jupyter Notebook

About This Project

This project presents a complete machine learning pipeline for heart disease prediction, implemented in a Jupyter notebook using the Heart Disease UCI dataset. The workflow begins with data cleaning and preprocessing, followed by an in-depth exploratory data analysis (EDA) to understand relationships between clinical features such as age, chest pain type, cholesterol level, and blood pressure with the target outcome. Data distributions and class imbalances were analyzed using histograms, bar plots, and correlation heatmaps to identify patterns and influential risk factors.

Multiple supervised learning models were trained and evaluated, including Logistic Regression, Decision Tree, Random Forest, Support Vector Machine (SVM), and K-Nearest Neighbors (KNN). Each model was assessed using standard classification metrics such as accuracy, precision, recall, F1-score, and confusion matrices, enabling a fair comparison of predictive performance. Visualization-driven evaluation helped interpret model behavior and highlight trade-offs between sensitivity and specificity.

The results demonstrate how classical machine learning techniques can be effectively applied to medical decision-support tasks. By comparing multiple models and performance metrics, the project emphasizes model interpretability, evaluation rigor, and data-driven decision making, making it a strong example of applied machine learning in healthcare analytics and a solid foundation for further research in medical AI.
